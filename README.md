# 🚀 Modern Job Listings Cards (React Props Demo)

A modern and responsive job listing cards UI built using **React.js** and **CSS**, focused on understanding and implementing **props handling in React**.

---

## 🚀 Live Demo

🔗 https://modern-job-listings-cards.vercel.app/

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 Vanilla CSS
- 📦 Lucide React Icons

---

## 🎯 Purpose of the Project

The main goal of this project is to:

- Understand how **props work in React**
- Learn how to pass data from **parent to child components**
- Practice building **reusable UI components**
- Render dynamic data using **JavaScript map()**

---

## ✨ Features

- 📄 Reusable Job Card Component
- 🔁 Dynamic Job Listings using Props
- 🎨 Clean & Modern UI Design
- 📱 Responsive Card Layout
- 💾 Save Button UI with Icon

---

## 🧠 What I Learned

- Passing multiple props efficiently
- Component reusability and clean structure
- Rendering lists dynamically in React
- Managing UI using simple and scalable CSS

---

## 📌 Key Implementation

- Job data is stored in an array inside `App.jsx`
- Each job is passed as props to the `Card` component
- Cards are dynamically rendered using `.map()`
- UI is built using pure CSS without any framework
